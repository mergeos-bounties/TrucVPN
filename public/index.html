Live exits come from mrgminner share. Offline sample catalog is used as fallback.

How it works

  1. Sharers run mrgminner share start and earn MRG for relayed residential traffic.
  2. TrucVPN dials a share exit (SOCKS5 / HTTP CONNECT).
  3. Apps use local SOCKS5/HTTP proxy — full-feature client without a kernel driver.
